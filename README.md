🎲 Bingo – Игра Бинго 
Классическая игра Бинго в терминале с генерацией карточек, автоматическим вызовом чисел и игрой против компьютера.
Отмечайте числа, закрывайте строки, столбцы и диагонали, чтобы победить!
Реализована на 7 языках программирования с единым интерфейсом и расширенными возможностями.

🚀 Возможности
Генерация карточек – стандартное поле 5×5 с числами от 1 до 75 (колонки B, I, N, G, O).

Центральная клетка FREE – автоматически отмечена.

Режимы игры:

Одиночная – игра против компьютера (вы и компьютер).

Два игрока – игра на одном устройстве по очереди.

Автоматический вызов чисел – ведущий случайно называет числа без повторений.

Автоматическая отметка – при вызове числа оно автоматически отмечается на всех карточках.

Проверка выигрыша – определяет, закрыта ли строка, столбец или диагональ.

Цветной вывод – отмеченные числа и победные линии подсвечиваются.

Статистика – количество побед и поражений сохраняется в файл.

Интерактивность – каждый ход отображает состояние карточек.

📖 Использование
Синтаксис (единый для всех версий):

bash
<команда> [режим] [опции]
Режимы
single – игра против компьютера (по умолчанию).

two – два игрока на одном устройстве.

Опции
Опция	Описание
-s, --stats	Показать статистику и выйти
-r, --reset	Сбросить статистику
-h, --help	Справка
Управление
В игре просто нажимайте Enter, чтобы продолжить (автоматический вызов чисел).

q – выход.

Примеры
bash
# Игра против компьютера
python bingo.py

# Два игрока
python bingo.py two

# Показать статистику
python bingo.py -s

🧠 Логика игры
Генерация карточек – для каждого игрока создаётся поле 5×5. Числа распределены по колонкам:

B: 1–15, I: 16–30, N: 31–45, G: 46–60, O: 61–75.

Центральная клетка (N-центр) – FREE (всегда отмечена).

Вызов чисел – из мешка (числа 1–75) случайно выбирается число, которое ещё не вызывалось.

Отметка – число отмечается на всех карточках, если оно там есть.

Проверка – после каждого хода проверяется, есть ли у кого-то закрытая строка, столбец или диагональ (бинго).

Победа – если бинго обнаружено, игра заканчивается, победитель объявляется.

Статистика – результаты сохраняются в локальный файл.

✨ Дополнительные фичи
Подсветка – выигрышная линия подсвечивается цветом.

Автосохранение – статистика сохраняется в ~/.bingo_stats.json.

Цветные карточки – отмеченные числа выделены зелёным, обычные – белым.
